Slope Relations

Fix spanning tree $T$ of $G$:

Each $f\not\in T$ gives an equation
$\small{\displaystyle\sum_{e\in C_T(f)}(m_e-m_f)(x_{e_1}-x_{e_2})=0}$

Create matrix $M_T$ with columns indexed by edges $e\in T$
Rows indexed by edges $f\not\in T$

entry in $(f,e)$-coordinate is $m_e-m_f$

$M_T\bar{x}=0$ if and only if valid drawing of graph
