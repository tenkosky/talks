Build Rigid Graphs

Observations:

1) A tree has $|E|=|V|-1$

2) If $G$ has a coupled spanning tree, then $|E|=2|V|-2$ (satisfies condition 1 of Laman)

3) If $G$ has a coupled spanning tree, then edges spread out (satisfies condition 2 of Laman)

Conclusion: $G$ is rigid if and only if it has a spanning subgraph with an almost coupled spanning tree*
